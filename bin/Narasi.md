Naming Convention adalah cara kita menamakan sebuah variabel agar variabel itu gampang dicari dan mengurangi kesalahan interpretasi dengan menggunakan nama sesuai fungsi variabel tersebut dengan aturan-aturan yang berlaku seperti dimulai dengan huruf kecil dan kata selanjutnya kapital sedangkan untuk kasus boolean harus diawali dengan pertanyaan lalu disusul dengan kata kerja
2. Penamaan variabel dan class dalam pemograman karena variabel merupakan tempat penyimpanan data yang memiliki identifier yang digunakan untuk mengakses dan memanipulasi data didalam sebuah program. Nama berfungsi sebagai identitas didalam penyimpanan. Sedangkan untuk class direpresentasikan sebagai sebuah blueprint dari suatu hal tertentu dalam sebuah sistem. sehingga penamaan variabel dan class yang baik memudahkan kita untuk mengetahui sebuah fungsi dari variabel dan class yang dibuat sehingga memudahkan kita dalam pengembangan dan pemeliharaan sebuah program.
3. Operator Aritmatika bekerja dengan mengambil nilai dari variabel kemudian melakukan operasi matematika sehingga menghasilkan nilai yang baru. 
Contoh operator aritmatika dalam java adalah : 
Penjumlahan (+), Pengurangan (-), Perkalian (*), Pembagian (/), Sisa Bagi / Modulus (%). 
Operator ini bekerja dengan memproses nilai dari variabel dan menghasilkan hasil sesuai operasi matematika.
